Welcome to the BMWPHD Backend


Horse Data

Get "/horses" to return all horses
GET "horses/search" with specifications in Request Body to find certain criteria matching horses

User Data

GET "/users" to return all users
GET "/users/customers" returns all customers
GET "/users/judges" returns all judges
GET "/users/{id}" returns a specific user
GET "/users/{id}/changeRequests" to return changeRequests a specific user has filled out
POST "/users" with user data in body to save a user
PUT "/users/{id}" with user data in body to update a user
DELETE "/users/{id}" to delete a user from database

ChangeRequest Data

GET "/changeRequests" returns all changeRequests
GET "/changeRequests/pending" returns all pending changeRequests GET "/changeRequests/{id}" returns a specific changeRequest
POST "/changeRequests" with changeRequest data in body to save a new changeRequest
PUT "/changeRequests/{id}" with changeRequest data in body to update changeRequest
DELETE "/changeRequests/{id}" to delete a specific changeRequest
